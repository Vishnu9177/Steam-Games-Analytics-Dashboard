# Steam-Games-Analytics-Dashboard
An end-to-end Steam Games Analytics project using Python, MySQL, and Power BI. The project analyzes game ratings, pricing, genres, platforms, ownership, and player engagement through data cleaning, SQL analysis, and interactive dashboards to generate actionable business insights.

# 📊 Key Features
* 🧠 Executive Summary
   * Total games, ownership, approval ratings, price
   * Genre Word Cloud, platform distribution
   * Top 15 highest-rated games by community score
*  📈 Trends Over Time
   * Game release trends by year
   * Playtime, rating, and pricing evolution
   * Ownership trends via stacked columns
* 🎮 Game Explorer
   * Slicers for genre, platform, rating, price
   * Drillable table with custom metrics
   * Treemap by genre
* Top 10 Games visualized via Bookmarks:
   * 🔝 Value Score (Hrs/€)
   * 🕹️ Average Playtime
   * ⭐ Average Rating
   * 🔙 Easily switchable with a Back to Overview button
 
# 🧰 Tools Used
* Power BI Desktop
* Power Query (M)
* DAX Measures
* Custom Steam-style JSON Theme
